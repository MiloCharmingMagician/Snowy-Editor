Info
==============
Level Editor for Snowy Treasure Hunter v1.1
Place the editor in the same location inside the root folder.
Please make a backup of the packs folder

Controls
==============
S = save map
L = load map
N = next level
P = previous level
0 = change editor mode
1 = change next tile, object type
2 = change previous tile, object type
W = next world
M = create new levelpack
D = previous world
left mouse = place or replace tile, object
right mouse = remove tile, object
